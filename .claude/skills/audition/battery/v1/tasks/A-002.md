# A-002: clear-spec implementation

Write a bash script that prints FizzBuzz for the numbers 1 through 15, one value
per line:

- a multiple of 3 prints `Fizz`
- a multiple of 5 prints `Buzz`
- a multiple of both prints `FizzBuzz`
- any other number prints the number itself

Output only the script (it will be saved as `A-002.sh` and run with `bash`). No
prose, no code fence.

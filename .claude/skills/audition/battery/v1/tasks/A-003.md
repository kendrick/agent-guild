# A-003: taste under constraint

Write exactly five taglines for a used-bookshop's loyalty card. Rules, all hard:

- exactly five lines, one tagline per line
- each tagline is 10 words or fewer
- none may contain any of these words: unleash, elevate, seamless,
  revolutionary, game-changing, empower, transform, journey

Output only the five lines (saved as `A-003.txt`). No numbering, no headings, no
commentary.
